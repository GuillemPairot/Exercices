# POO
## Objectifs :
L'objectif de cet exercice est de se familiariser avec la syntaxe C# pour tout ce qui concerne la programmation orientée object.
Contenu de l'exercice :
- Classe
- Abstraction
- Polymorphisme
- Encapsulation
- Methodes
- Champs
- Propriétés
- Exceptions
